NoteBin

NoteBin is a mobile application built with Flutter that allows users to create, edit, and manage their notes. It leverages Firebase Cloud Firestore for cloud-based storage, making notes accessible from any device.

Features

  • Create, Edit, and Delete Notes: Effortlessly add, modify, and remove notes as needed.
  • Cloud-Based Storage: Notes are securely stored in Firebase Cloud Firestore, ensuring accessibility across devices.
  • User-Friendly Interface: The intuitive interface provides a seamless user experience for managing notes.
